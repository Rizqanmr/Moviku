# Moviku
A movie app using API from [TMDB](https://developer.themoviedb.org/docs/getting-started)

## Feature
branch: master
- Discover movies based on categories
- Detail movie
- YouTube player for movie trailer
- User review

branch: version-2
- Implement offline mode with local database (Room)
- Implement paging
- Sort ascending and descending
- Search for a movie based on a list

## Notes
- Download this project
- Open this project with Android Studio
- Before building and running this project. Go to local.properties and add 2 lines of code:

```
BASE_URL = "https://api.themoviedb.org/3/"
API_KEY = "Enter your themoviedb personal access token"
```

- Sync Project with Gradle File
- Run
